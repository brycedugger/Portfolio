Moolah

Moolah is a full-stack application that helps users secure financial peace of mind. Using our app, the user can track their expenses and goals per category. The user can view their custom, input expenses, goals, and categories in tabular view to help develop a budget that works for them.

Built with:

  • Front End: jQuery, Bootstrap / CSS

Light Snow

Light Snow is a webpage that displays the weather conditions of various mountain resorts around the country with a light twist. The webpage uses an API to connect a Philips Smart LED lightbulb to get visual notifications of weather conditions at specified mountain resort. If the user logs in, they may add a resort if they know it's latitude and longitude.

The user can select a resort from the drop down menu to display the:

  • Icon of Weather Conditions and Lightbulb Color Reaction
  • Local Time
  • Description of Weather Conditions
  • Temperature (Celsius or Fahrenheit based on NavBar Toggle
  • Wind Speed
  • Wind Direction
  • Location

Built with:

  • Front End: jQuery, Bootstrap / CSS
  • Database: Firebase / Local Storage
  • Authentication: Google Authenticator
  • APIs:
    • OpenCage Geocoder
    • OpenWeatherMap
    • LIFX
    • Google Maps
